# DevSecOps Lab

## Project Overview

This project demonstrates a basic DevSecOps pipeline using:

- GitHub Actions
- Docker
- Trivy Security Scanner
- Linux (WSL Ubuntu)

## Workflow

Git Push
↓
GitHub Actions
↓
Docker Build
↓
Trivy Security Scan
↓
Success

## Security Features

- Container vulnerability scanning using Trivy
- Non-root container execution
- SSH authentication for GitHub access

## Technologies Used

- Python Flask
- Docker
- GitHub Actions
- Trivy
- Linux
